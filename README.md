# 项目介绍
> 当前项目会一步一步带大家写代码，和分析较为安全的NFT项目源码等。

当前项目是一个`全栈开发以太坊ERC721协议的全套NFT项目`，其中内容是从`NFT基础概念`、 `NFT相关内容介绍`、 `各种类型NFT开发`、 `NFT项目部署`、`前端web3常用库` `前端NFT开发`、 `前端部署多种环境`、 `NFT安全`等集一体，可以快速带大家学会`整套完整NFT开发`的一个项目
